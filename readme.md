
# MEMORIA - веб альбом воспоминаний
Вы смазливо-противная парочка нетакусь? Группа кентов с кучей историй и трофейных фото с тусовок? Труппа скуфяр, которые хотят друг перед другом покичится прекрасными *прострелами* и навсегда приберечь во времени подвиги на передовой? И при этом, все это дело неорганизованно хранить в какой-нибудь конфе вам неудобно? Муторно оглядываться назад и припоминать прошедшее?
### Тогда милости прошу - MEMORIA к вашим услугам.
## Локальный селфхост
Приложение для локального селфхостинга MEMORIA - [MEMORIA Host Manager](https://github.com/DraSolace/Memoria-host-manager)
## Функционал веб приложения:

 - Grid система с виджетами под записки (richtext) и фотографии (в base64)
 - Динамичекий ресайзинг и drag-n-drop виджетов
 - Разделители. Их можно свернуть, а вместе с ними и всё что находится под ними, вплоть до следующего разделителя. А так же они являются "закладками", к которым можно прыгнуть из навигации
 - Hero секция с каруселью рандомных записок и фотографий, а так с рандомной ротацией фраз, список которых можно пополнить из футера.
 - API для централизованного хранения данных (очевидно). IndexedDB не используется 
 - Всякая анимированная шелуха, чтоб зрыкалам приятно было.
## Установка и селфхост в локальной сети
1. Внутри папки с сурсом:
> pnpm install

> pnpm build

2. В установщике [MEMORIA Host Manager](https://github.com/DraSolace/Memoria-host-manager):

* Поставить домен
    * или использовать IPv4:80
* Установить сам `Host Manager`
    * или скачать его релиз

3. Затем запустить `Host manager` и выбрать корневую папку с MEMORIA.

 *Сайт становится доступен всем устройствам в локальной сети, что добавили себе домен по MEMORIA:80 (или `{IPv4 Хоста}`:80, если домены решили не добавлять)*
